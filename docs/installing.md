# 安装指南

## 系统要求

- Python 3.10+
- SQLAlchemy 2.0+
- Pydantic 2.0+

## 安装

```bash
pip install sqlalchemy-crud-plus
```

### 版本兼容性

确保 SQLAlchemy 版本为 2.0+：

```bash
pip install --upgrade sqlalchemy>=2.0.0
```
