# 安装指南

## 系统要求

- Python 3.10+
- SQLAlchemy 2.0+
- Pydantic 2.0+

## 安装

=== "pip"

    ```bash
    pip install sqlalchemy-crud-plus
    ```

=== "uv"

    [uv](https://github.com/astral-sh/uv) 是一个极快的 Python 包管理器，推荐使用。

    安装包：

    ```bash
    uv pip install sqlalchemy-crud-plus
    ```

    或者添加到项目依赖：

    ```bash
    uv add sqlalchemy-crud-plus
    ```
